Personal Name Colourization

The analyzed personal names are bracketed using Unicode SQUARE BRACKET WITH QUILL (⁅=U+2045 and ⁆=U+2046). They are colourized with the following correspondences:

  • BLUE = Name of principal (masculine or unmarked)
  • PINK = Name of principal (feminine)
  • PURPLE = Parent's name
  • ORANGE = Grandparent's name
  • RED = Clan name
  • NAVY = Spouse name
  • OLIVE = name link element
  • BROWN = roleName element

Other Annotated Features

  • TEAL = PLACE
  • MAROON = DATE
  • ⚓: an anchor shows the head of a group of siblings, where head is defined as the right-most member of the group, followed by the additional naming information
  • ⊳: a right-pointing triangle shows membership of the named person in a sibling-group

Discourse Sidebar Colourization

The various discourse blocks are indicated with the coloured bar down the left margin. The correspondences are:

  • BLACK = BODY
  • RED = WITNESSES
  • GREEN = COLOPHON
  • BLUE = SEALINGS
  • PURPLE = SUMMARY
