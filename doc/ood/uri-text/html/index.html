Introduction

Oracc's stable URIs are formed according to simple patterns. This document demonstrates the patterns which are used for requesting different parts and forms of texts.

Pattern: PROJECT/NUMBER

/cams/gkab/P338652

/saao/knpp/P223388

HTML facing-page transliteration and translation, with pager.

Pattern: PROJECT/NUMBER,NUMBER ...

/cams/gkab/P338652,P338628

/saao/P223388,P334194

Item list with pager.

Pattern: PROJECT/NUMBER/html

/cams/gkab/P338652/html

/saao/P223388/html

HTML facing-page transliteration and translation, no pager.

Pattern: PROJECT/NUMBER/translit

/cams/gkab/P338652/translit

/saao/P223388/translit

HTML transliteration only.

Pattern: PROJECT/NUMBER/translation

/cams/gkab/P338652/translation

/saao/P223388/translation

HTML translation only.

Pattern: PROJECT/NUMBER/score

/dcclt/Q000001/score

Full HTML score.

Pattern: PROJECT/NUMBER/cuneified

/saao/saa01/P334194/cuneified

Cuneified transliteration.

Pattern: PROJECT/NUMBER/proofing

/etcsri/Q001253/proofing

Proofing output (beta).

Pattern: PROJECT/NUMBER/tei

/hbtin/P342428/tei

TEI version of text served as CSS-styled XML.

Pattern: PROJECT/NUMBER/cat

/hbtin/P342428/cat

HTML version of catalogue data.

Pattern: PROJECT/NUMBER/xml

/hbtin/P342428/xml

XML version of text data in Oracc markup.

Pattern: PROJECT/NUMBER/cat/xml

/hbtin/P342428/cat/xml

XML version of catalogue data in Oracc markup.

Pattern: PROJECT/NUMBER/translit/xml

/hbtin/P342428/translit/xml

XML version of transliteration in Oracc markup.

Pattern: PROJECT/NUMBER/translation/xml

/hbtin/P342428/translation/xml

XML version of translation in Oracc markup.

Pattern: PROJECT/NUMBER/score/xml

/dcclt/Q000001/score/xml

XML version of score in Oracc markup.

Pattern: cdli/NUMBER/images

/cdli/P313915/images

HTML with links to all CDLI images for the text.

Pattern: cdli/NUMBER/image

/cdli/P000001/image

Wrapper returning best available CDLI image.

Pattern: cdli/NUMBER/thumb

/cdli/P000001/thumb

Wrapper returning best available CDLI thumbnail.
